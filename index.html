Tii Software Studio

Tii Software Studio (TiiSoft) is the team behind the Tii Framework - a high-performance programming framework for developing Web applications in PHP 5.

Members of TiiSoft are all veteran Web application developers holding master or higher degrees on computer science or related fields.

TiiSoft can offer to your organization professional support on training and using the Tii framework. It also offers to develop your custom Web application projects.
